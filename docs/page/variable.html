Variables

Variables store data that can be accessed and modified. They are composed with a type and an identifier. They can be initialized to a value when declared.

The type of variable can be inferred by using the auto keyword in place of the type. These variables will take the type of their initial value.

Dynamic variables do not have a set type, they can change as needed.

Definition

type identifier

type identifier = value

type x, y, z

type a, b = value

auto identifier

auto identifier = value

auto identifier = (auto) value

Examples

/* ========== Example 1 ========== */

int x = 5
int y = 7, z = 9
float pi = 3.14
string str = "Hello World"

/* ========== Example 2 ========== */

auto a = 123
auto b = 3.14
auto c = "Hello World"
	
/* ========== Example 3 ========== */

auto a
auto b = (auto) 5
