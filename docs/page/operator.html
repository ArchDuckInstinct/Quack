Operators

Operators modify variables. They can get the sum, difference, product, etc of two numbers, concatenate strings, and so on.

When using an operator on 2 different data types, generally the data type that is more capable of representing both values is the result.

Examples

/* ========== Example 1 ========== */

int c = a + b
int d = c * 5
int e

e = (a + b) * 5

auto f = ++e

/* ========== Example 2 ========== */

string s = "Hello World!"
int i = 5

auto result = s + " " + i

Data Types

Operator Name Description
= Assign Directly sets the value of a variable.
Arithmetic Applies mathmetical operations to numbers. These can be applied between bools, ints, and floats. When mixing types, false bool values will become 0 (int or float), true bool values will become 1 (int or float). Ints will be converted to floats.
+ Addition Adds two numbers together, returning the sum.
- Subtraction Subtracts one number from another, returning the difference.
* Multiplication Multiplies two numbers together, returning the product.
/ Division Divides one number from another, returning the quotient.
% Modulus Divides one number from another, returning the remainder.
** Exponent Raises one number to the power of another.
Unary Unary operators only work with one single variable instead of two. Some unary operators have prefix and postfix versions. The difference is prefix versions will return the value after the operation while postfix will return the value before the operation.
++ (prefix & postfix) Increment Adds 1 to a number.
-- (prefix & postfix) Decrement Subtracts 1 from a number.
! (prefix) Not Inverts a boolean value. False becomes true and true becomes false.
? (postfix) Not Inverts a boolean value. False becomes true and true becomes false.
Comparison
== Equal
!= Not Equal
?= Threshold Equals Is number equal to value within a threshold. Useful for floating point values.
a ?= 5,0.1
>< Range Equals Is number equal within a min/max range.
a >< 3,5
