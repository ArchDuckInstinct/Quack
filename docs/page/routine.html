Routines

Routines are Quack's analog to functions, they are comprised of an optional return type, an identifier, an optional list of parameters, and a block of statements. Routines can be called elsewhere in the code.

Definition

type identifier()
	statement...
			
type identifier(type parameter...)
	statement...

Examples

/* ========== Example 1 ========== */

void main()
	write("Quack!!!")
	
/* ========== Example 2 ========== */

int getSum(int a, int b)
	int result = a + b
	
	return result
	
/* ========== Example 3 ========== */

void yetAnotherRoutine(int a)
	string str = "The numbers are "
	int b = getSum(a, 3)
	a += 5
	
	if a > 0
		write(a + " is greater than 0\n")
	
	if a > 10
		write(a + " is greater than 10\n")
	
	str += a + ", " + b
	write(str)
